# Odin - Testing Practice (Jest)

![Status Badge](https://img.shields.io/badge/status-concluded-brigthgreen?style=for-the-badge)

This project was built to practice **Test-`Driven Development (TDD)** as a part of the **Testing Javascript** curriculum from [The Odin Project](https://www.theodinproject.com).

The project includes the implementation of several JavaScript utilities, each developed following the TDD workflow: writing tests first, implementing the functionality, and refactoring the code while keeping all tests passing.

- `capitalize` – Capitalizes the first letter of a string.
- `calculator` – Provides add, subtract, multiply, and divide methods.
- `caesarCipher` – Encrypts text using the Caesar cipher with a configurable shift.
- `analyzeArray` – Analyzes an array of numbers and returns an object containing the average, minimum value, maximum value, and length.

Each feature is fully covered by unit tests written with Jest.

## 🛠️ Tech Stack

- JavaScript
- Jest

## 🚀 Installation and Usage

Clone the repository:

```bash
git clone https://github.com/guisalva/odin-testing-practice.git

cd odin-testing-practice
```

Install the dependencies:

```bash
npm install
```

Run all the tests:

```bash
npm test
```
